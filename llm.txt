## ⚡ TECHNICAL STANDARDS
1. **Images**: MUST be WebP. Use `tools/optimize_media.py`.
2. **Assets**: CSS/JS MUST be minified. Use `tools/minify_assets.py`.
3. **Links**: Use `.min.css` and `.min.js` in Production.
4. **Data**: Hybrid JSON loading (fetch 'media/projects.json' w/ fallback).
5. **Schema**: EVERY page needs JSON-LD (BreadcrumbList + Context).

## 🚀 DEPLOYMENT PROTOCOL
1. **Changes**: Modify code locally.
2. **Minify**: Run minification scripts if CSS/JS changed.
3. **Push**: `git push origin main`.
4. **Deploy**: Trigger server pull via SSH (see `ssh_connection/SSH_GUIDE.md`).
