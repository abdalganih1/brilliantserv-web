# 🏗️ مؤسسة روائع الكريستال - نظام عرض القوالب

نظام عرض قوالب متعددة لموقع مؤسسة روائع الكريستال للأعمال الكهربائية والميكانيكية.

---

## 📁 هيكلية المشروع

```
BrilliantServ/
├── 📄 index.html                    # صفحة اختيار القوالب الرئيسية
├── 📁 templates/                    # مجلد القوالب
│   ├── 📁 template-1/               # القالب الأزرق الداكن
│   ├── 📁 template-2/               # القالب الأخضر (Dark Neon)
│   └── 📁 template-3/               # القالب الأخضر الفاتح ⭐ (الموصى به)
├── 📁 media/                        # ملفات الوسائط الأصلية
├── 📁 css/                          # ملفات CSS الأصلية
├── 📁 js/                           # ملفات JavaScript الأصلية
└── 📁 ملفات المستخدم/               # ملفات PDF المستخرجة
    ├── 📁 شعارات العملاء/           # 11 شعار عميل
    ├── 📁 شهادات/                   # 20 شهادة ووثيقة
    ├── 📁 مراجعنا/                  # 15 شعار مرجع
    └── 📁 extracted/                # الصور المستخرجة من PDF
```

## 🎨 القوالب المتاحة

| # | اسم القالب | الألوان | الحالة | الوصف |
|---|------------|---------|--------|-------|
| 1 | الأزرق الداكن | 🔵 أزرق + ذهبي | ✅ جاهز | التصميم الأصلي المستورد |
| 2 | الهوية الخضراء | 🟢 أخضر نيون + برتقالي | ✅ جاهز | تصميم من هوية PDF |
| 3 | **الأبيض العصري** | ⚪ أبيض + أخضر | ✅ **جاهز** | تصميم حديث ونظيف |

## 🚀 تشغيل المشروع

## 🎯 التركيز على الخدمات

حسب أوامر العميل، الموقع يركز على:

✅ **المضخات والبوسترات**
- مضخات الضغط (Booster Pumps)
- مضخات المياه المنزلية والصناعية
- مضخات الحريق
- محطات الضخ المتكاملة

✅ **لوحات التحكم الكهربائية**
- لوحات التحكم الرئيسية (MCC)
- لوحات التوزيع (Distribution Panels)
- لوحات التحكم الآلي (PLC)
- لوحات التشغيل للمضخات

❌ **تم استبعاد**: مضخات سكالا، مضخات خرسانة، الكترونيات، صيانة بوردات
