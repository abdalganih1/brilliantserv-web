## 📊 نظام البيانات (Hybrid JSON Loading)

### كيف يعمل؟

الموقع يستخدم **نظام تحميل هجين** للبيانات:

1. **مع سيرفر (استضافة)**: يحمّل البيانات من ملفات JSON المنفصلة
2. **محلياً (file:///)**: يستخدم البيانات المضمنة في HTML كـ fallback

```javascript
// Hybrid: Try fetch, fallback to embedded
try {
    const response = await fetch('media/projects.json');
    data = await response.json();
    console.log('✓ Loaded from JSON');
} catch (e) {
    data = embeddedData;
    console.log('✓ Using embedded data (file:// mode)');
```

## ✨ الميزات التقنية

### التحميل التدريجي للصور (Progressive Loading)
- تظهر placeholder ملونة فوراً
- الصور تُحمّل في الخلفية مع `loading="lazy"`
- تأثير fade-in عند اكتمال التحميل

### الفلترة الديناميكية
- فلترة المشاريع حسب الفئة (كهربائي/ميكانيكي/لف محركات)
- دعم URL hash للروابط المباشرة (`#electrical`, `#mechanical`, `#winding`)
- النقر على أي مشروع يُفلتر حسب فئته

### التوافق
- يعمل مباشرة بـ `file:///` بدون سيرفر
- يعمل مع أي استضافة ويب
- متوافق مع جميع المتصفحات الحديثة
